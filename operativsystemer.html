Operative Systemer

Teknologien er nået langt

De væsentligste træk ved den computer, vi kender i dag, blev fastlagt omkring 1945. Forud var gået en 100-årig udvikling af mekaniske og stadig mere automatiske regnemaskiner samt en udvikling af mekaniske og elektriske databehandlingsmaskiner som fx hulkortmaskiner.

Windows

Microsoft Windows er en serie af software operativsystemer og grafiske brugerflader, som produceres af den amerikanske virksomhed Microsoft. Den første udgave af systemet hed Windows 1.0 og blev lanceret den 20. november 1985, som en grafisk brugerflade til operativsystemet MS-DOS, som Microsoft havde købt i 1980 og videreudviklet på siden.

Interne Disk

Mac indeholder en intern disk, som er en lagringsenhed, der indeholder de apps og oplysninger, som Mac bruger. Nogle Mac-computere har yderligere interne diske eller tilsluttede eksterne diske.

Startdisken indeholder macOS-systemsoftware. Det er typisk den interne disk, der er startdisk, men det er også muligt at starte Mac fra en netværksdisk eller ekstern disk med systemsoftware.
